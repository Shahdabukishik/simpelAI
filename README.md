# Simple AI Image Classifier

This is a very simple AI project using Python and scikit-learn. 
It classifies images into different categories (example: cats and dogs).

## Libraries
- numpy
- scikit-learn
- scikit-image

## How to run
1. Install the required libraries:
   pip install -r requirements.txt
2. Add your images to the `images/` folder.
   Name them like `label_1.jpg`, `label_2.png`, etc.
3. Run the project:
   python classify_images.py

## Purpose
- Demonstrates basic AI concepts using a ready-made dataset
- Allows adding custom images for classification
- Easy to understand and extend
